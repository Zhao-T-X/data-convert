# DataConvert - utools 数据转换插件

**日期**: 2026-04-26
**版本**: v1.0

## 概述

一个 utools 插件，用于快速将 Excel/CSV/JSON 数据转换为程序可用的格式。

## 核心功能

### 输入
- **剪贴板粘贴**: Ctrl+V 直接粘贴数据
- **文件拖入**: 支持 .xlsx .xls .csv .json

### 输出格式
- **JSON**: 数组对象格式，API 开发常用
- **SQL**: INSERT 语句，数据库导入
- **CSV**: 逗号分隔，Excel 兼容
- **YAML**: 配置文件格式

### 性能
- 支持最大 10 万行数据

## 技术方案

- **前端**: Vue 3 + Vite
- **文件解析**: xlsx (Node.js)
- **打包**: vite-plugin-utools-helper
- **输出格式**: 纯原生实现
